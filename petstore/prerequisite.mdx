---
title: 'Prerequisite'
description: 'Before you begin, ensure that your development environment meets the following requirements'
---

Follow these steps to install and run Zillabase on your operating system:

**1. Homebrew installed**

Install [Homebrew](https://brew.sh/) if it's not already installed. 

Verify installation by running :

```bash
brew --version
```

**2. Java installed**

Install [Java](https://www.oracle.com/java/technologies/downloads/) if it's not already installed. 

Verify installation by running :

```bash
java --version
```

**3. Docker installed and running**

Install [Docker](https://www.docker.com/products/docker-desktop/) if it's not already installed.

Verify docker running with :

```bash
docker info
```
